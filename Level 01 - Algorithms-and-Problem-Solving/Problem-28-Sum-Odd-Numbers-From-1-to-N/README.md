# Sum Odd Numbers From 1 to N



## 🧠 Problem Description
This program calculates the sum of all odd numbers from $1$ up to a user-defined number $N$. It demonstrates the solution using three types of loops: `while`, `do...while`, and `for`.

## ⚙️ Solution Overview
- **ReadNumber()**: Captures the limit $N$ from the user.
- **enOddOrEven**: An enumeration to clearly define whether a number is Odd or Even.
- **CheckOddOrEven()**: A helper function that uses the modulo operator `%` to determine the parity of a number.
- **SumOddNumbers Functions**: Three separate functions that iterate through the range, check for odd numbers, and accumulate the sum.

## 🛠️ Key Concepts
- Enumerations (`enum`)
- Modulo Operator (`%`)
- Accumulator Pattern (`Sum += counter`)
- Function Reuse and Modular Logic
- Loop Comparison
