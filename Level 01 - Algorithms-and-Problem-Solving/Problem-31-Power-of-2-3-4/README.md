# Power of 2, 3, 4



## 🧠 Problem Description
This program takes a number as input and calculates its second, third, and fourth powers (i.e., $n^2, n^3, n^4$).

## ⚙️ Solution Overview
- **ReadNumber()**: Captures the base number from the user.
- **PowerOf_2_3_4()**: Calculates the powers by multiplying the number by itself for each level and prints the results separated by spaces.
- **main()**: Directly passes the input function to the processing function for a concise execution.

## 🛠️ Key Concepts
- Basic Arithmetic (Multiplication)
- Power calculation logic
- Function-based programming
- Minimalist code design
