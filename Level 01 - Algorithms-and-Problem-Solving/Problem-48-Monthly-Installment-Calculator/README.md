# Monthly Installment Calculator

## 🧠 Problem Description
This program calculates the required monthly installment amount to settle a loan, given the total loan amount and the desired number of months for repayment.

## ⚙️ Solution Overview
- **ReadPositiveNumber()**: Ensures that the loan amount and the number of months are positive values to maintain mathematical validity.
- **CalculateMonthlyInstallment()**: Implements the formula: $\text{Monthly Installment} = \frac{\text{Loan Amount}}{\text{Total Months}}$.
- **main()**: Collects user input and displays the calculated monthly payment.
