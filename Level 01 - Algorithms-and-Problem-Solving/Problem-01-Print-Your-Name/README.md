# Problem 01: Print Your Name

## 📝 Description
A simple C++ program that demonstrates how to use functions to display a string (name) on the console. This is the foundational step in learning C++ function structure.

## 💡 Logic / Approach
- Create a `void` function named `PrintName` that takes a `string` parameter.
- Use `std::cout` inside the function to format and print the name.
- Call the function from the `main()` scope with a specific string argument.
