# Read First and Last Name and Print Full Name

## 🧠 Problem Description
This C++ program reads a user's **first name** and **last name**, then combines them to display the user's **full name**.

## ⚙️ Solution Approach
- A `struct` is used to store the user's first and last names.
- **ReadInfo()**: Reads the first name and last name from the user.
- **GetFullName()**: Combines the first and last names into a single string.
- **PrintFullName()**: Prints the full name in a formatted output.
