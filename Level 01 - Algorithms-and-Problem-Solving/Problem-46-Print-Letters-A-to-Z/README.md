# Print Letters A to Z

## 🧠 Problem Description
This program prints all uppercase letters from **A** to **Z** using their corresponding ASCII values.

## ⚙️ Solution Overview
- **ASCII Representation**: In C++, every character has a numeric code. The uppercase 'A' starts at **65**, and 'Z' ends at **90**.
- **PrintLettersAtoZ()**: 
    - Uses a `for` loop to iterate from 65 to 90.
    - Inside the loop, it casts the integer `i` to a `char` type to display the actual letter.
- **main()**: Calls the function to execute the printing process.

## 🛠️ Key Concepts
- ASCII Character Encoding
- Type Casting (`int` to `char`)
- For Loop iteration
- Character manipulation in C++
