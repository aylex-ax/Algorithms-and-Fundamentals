# Sum of Three Numbers

## 🧠 Problem Description
This program reads three integers from the user and calculates their total sum.  

The goal is to demonstrate input/output operations, arithmetic calculations, and function-based task separation in C++.

## ⚙️ Solution Overview
- **ReadNumbers()**: Reads three integers from the user using reference parameters.  
- **SumOf3Numbers()**: Calculates the sum of the three numbers.  
- **PrintResult()**: Outputs the total sum.  
- Functions are used to separate tasks, improving code clarity and maintainability.

## 🛠️ Key Concepts
- Pass by reference  
- Arithmetic operations  
- Function-based code organization  
- Standard input/output operations
