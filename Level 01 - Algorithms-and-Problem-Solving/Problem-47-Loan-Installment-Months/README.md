# Loan Installment Months Calculator



## 🧠 Problem Description
This program calculates the total number of months required to settle a loan based on the total loan amount and the fixed monthly installment provided by the user.

## ⚙️ Solution Overview
- **ReadPositiveNumber()**: A validation function that ensures both the loan amount and the installment are positive values to avoid logical or mathematical errors (like division by zero).
- **CalculateTotalMonths()**: Implements the formula: $\text{Total Months} = \frac{\text{Loan Amount}}{\text{Monthly Installment}}$.
- **main()**: Collects the data and displays the total months needed for full repayment.

## 🛠️ Key Concepts
- Financial Logic (Debt Amortization basics)
- Input Validation
- Floating-point Arithmetic
- Function-based Structure
