# Maximum of Two Numbers

## 🧠 Problem Description
This program reads two numbers from the user and determines the maximum value between them.

The purpose of this problem is to demonstrate basic comparison logic, input/output handling, and clean function-based program structure in C++.

## ⚙️ Solution Overview
- The program reads two integers from the user.
- A comparison is performed to determine the larger number.
- The result is printed in a clear and readable format.
- Functions are used to separate responsibilities and improve code clarity.

## 🛠️ Key Concepts
- Comparison operators  
- Conditional logic  
- Pass by reference  
- Function-based program organization  
- Standard input/output
