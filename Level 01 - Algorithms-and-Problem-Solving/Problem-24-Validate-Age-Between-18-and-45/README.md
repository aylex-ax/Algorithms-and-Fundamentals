# Validate Age Between 18 and 45

## 🧠 Problem Description
This program prompts the user to enter their age and checks whether it falls within the valid range of 18 to 45 (inclusive).

## ⚙️ Solution Overview
- **ReadAge()**: Reads the user's age as a `short` integer.
- **ValidateNumberinRange()**: A reusable function that checks if a given number is between two specific limits.
- **PrintResult()**: Displays whether the age is "Valid" or "Invalid" based on the validation function's result.

## 🛠️ Key Concepts
- Boolean Functions
- Logical Operators (`&&`)
- Data validation
- Range checking
- Reusable function design
