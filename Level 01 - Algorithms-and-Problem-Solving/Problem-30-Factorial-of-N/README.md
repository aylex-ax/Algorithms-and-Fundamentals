# Factorial of N



## 🧠 Problem Description
This program calculates the factorial of a positive integer $N$ entered by the user. The factorial of $n$ (denoted as $n!$) is the product of all positive integers less than or equal to $n$.

Example: $5! = 5 \times 4 \times 3 \times 2 \times 1 = 120$.

## ⚙️ Solution Overview
- **ReadPositiveNumber()**: Ensures the user enters a non-negative integer using a `do...while` loop.
- **Factorial()**: Calculates the result by iterating from $N$ down to 1 and multiplying the values into a result variable.
- **main()**: Integrates the functions to deliver the final output.

## 🛠️ Key Concepts
- Factorial Mathematical Logic
- Input Validation (Positive numbers only)
- For Loops (Descending)
- Multiplication Accumulator
