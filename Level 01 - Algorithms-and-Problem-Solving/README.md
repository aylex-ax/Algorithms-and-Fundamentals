# Level 01 =----> (04)-Algorithms and Problem Solving Level 1
Part of a structured C++ learning roadmap ([ProgrammingAdvices.com](https://programmingadvices.com)) that builds programming fundamentals from the ground up — starting with basic I/O and functions, then moving through conditionals, loops, geometry-based math problems, and simple simulations like a basic ATM PIN system.

## 📂 Structure

Each problem lives in its own folder containing:
- A `.cpp` source file with the full solution
- A `README.md` explaining the problem, approach, and key concepts used
Level 01 - Algorithms-and-Problem-Solving/
├── Problem-01-Print-Your-Name/
├── Problem-02-Read-Name-Print-It/
├── ...
└── Problem-50-ATM-PIN-With-3-Tries/

## 🔑 Key Concepts Covered

- Functions, parameters, and return values
- Console input/output (`cin`, `cout`, `getline`)
- Conditional logic (`if`, `else if`, `switch`)
- Loops (`for`, `while`, `do-while`) and loop-based accumulation
- Basic geometry formulas (rectangles, triangles, circles, inscribed/circumscribed shapes)
- Prime number checking and factorial calculation
- Simple financial calculations (commission, tax, loan installments)
- Early state-based simulation logic (PIN login with limited attempts)

## 🛠️ Requirements

- Any standard C++ compiler (g++, MSVC, Clang) supporting C++11 or later
- No external libraries required

## ▶️ How to Run

```bash
g++ ProblemFile.cpp -o Problem
./Problem
```

## 📌 Notes

This level is the entry point of the roadmap — it builds the core programming reflexes (I/O, conditionals, loops, functions) that every later level, including matrix handling, string processing, and file-based data storage in Level 03, builds on top of.
