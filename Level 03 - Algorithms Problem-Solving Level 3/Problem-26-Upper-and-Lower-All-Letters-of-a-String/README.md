# Problem 26: Upper & Lower All Letters of a String

## 📋 Description
Reads a string and converts it fully to uppercase, then fully to lowercase.

## 💡 Solution Overview
- `UpperAllSrting()` converts every lowercase letter to uppercase
- `LowerAllString()` converts every uppercase letter to lowercase

## 🔑 Key Concepts
- Full-string case conversion
- `islower()` / `isupper()` guards before converting

## 📥 Sample Output
```
String After Upper :
AHMED YASER

String After Lower :
ahmed yaser
```
