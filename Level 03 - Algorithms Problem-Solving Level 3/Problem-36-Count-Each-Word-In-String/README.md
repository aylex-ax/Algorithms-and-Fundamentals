# Problem 36: Count Each Word In String

## 📋 Description
Reads a sentence and counts how many words it contains.

## 💡 Solution Overview
- `CountWords()` uses the same find/substr/erase loop as Problem 35, but increments a counter instead of printing

## 🔑 Key Concepts
- Reusing a parsing pattern for a different output (count vs. print)

## 📥 Sample Output
```
The Number of words in your string is : 3
```
