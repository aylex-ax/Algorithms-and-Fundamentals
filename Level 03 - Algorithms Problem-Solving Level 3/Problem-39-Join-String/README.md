# Problem 39: Join String

## 📋 Description
Joins all elements of a `vector<string>` into a single string, separated by a delimiter.

## 💡 Solution Overview
- `JoinString()` appends every word plus the delimiter, then trims the trailing delimiter off the end

## 🔑 Key Concepts
- The inverse operation of Split (Problem 37)
- Range-based for loop over a vector

## 📥 Sample Output
```
Vector after Join :
Ahmed Yaser Metwalli Mohamed Mohamed Daoud
```
