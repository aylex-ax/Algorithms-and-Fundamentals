# Problem 27: Invert Character Case

## 📋 Description
Reads a single character and flips its case (upper → lower, lower → upper).

## 💡 Solution Overview
- `InvertLetterCase()` uses a ternary: if the character is uppercase, lowercase it; otherwise uppercase it

## 🔑 Key Concepts
- Ternary operator
- `isupper()`, `toupper()`, `tolower()`

## 📥 Sample Output
```
Please enter a Character ? a
Character After Inverting Case :
A
```
