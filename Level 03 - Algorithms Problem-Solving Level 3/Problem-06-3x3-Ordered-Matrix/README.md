# Problem 6: 3x3 Ordered Matrix

## 📋 Description
Fills a 3x3 matrix with sequential numbers from 1 to 9 and prints it.

## 💡 Solution Overview
- `FillMatrixWithOrderedNumbers()` uses a counter that increments on every cell

## 🔑 Key Concepts
- 2D arrays
- Counter variables inside nested loops

## 📥 Sample Output
```
  1   2   3
  4   5   6
  7   8   9
```
