# Problem 14: Check Scalar Matrix

## 📋 Description
Checks whether a given 3x3 matrix is a scalar matrix (same constant value on the diagonal, 0s elsewhere).

## 💡 Solution Overview
- `IsScalarMatrix()` captures the first diagonal element, then checks all diagonal cells match it and all off-diagonal cells are 0

## 🔑 Key Concepts
- Diagonal detection
- Generalizing Problem 13 (fixed value 1 → any constant)

## 📥 Sample Output
```
Yes: Matrix is Scalar
```
