# Problem 7: Transpose Matrix

## 📋 Description
Builds an ordered 3x3 matrix, then computes and prints its transpose (rows become columns).

## 💡 Solution Overview
- `TransposeMatrix()` copies `Arr[j][i]` into `ArrTransposed[i][j]`

## 🔑 Key Concepts
- Matrix transposition
- Index swapping between two matrices

## 📥 Sample Output
```
Original:      Transposed:
1  2  3         1  4  7
4  5  6         2  5  8
7  8  9         3  6  9
```
