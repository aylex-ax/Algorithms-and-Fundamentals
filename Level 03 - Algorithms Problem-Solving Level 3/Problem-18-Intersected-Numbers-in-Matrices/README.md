# Problem 18: Intersected Number in Matrices

## 📋 Description
Prints every number from Matrix 1 that also appears somewhere in Matrix 2.

## 💡 Solution Overview
- `PrintIntersectedNumbers()` loops over Matrix 1 and uses `IsNumberInMatrix()` to check each value against Matrix 2

## 🔑 Key Concepts
- Combining two matrices with a helper search function
- Set-intersection style logic on arrays

## 📥 Sample Output
```
Intersected Numbers are :
5   22   1
```
