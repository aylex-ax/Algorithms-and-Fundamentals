# Problem 8: Multiply 2 Matrices

## 📋 Description
Fills two 3x3 matrices with random numbers and multiplies them element by element (Hadamard-style, not true matrix multiplication).

## 💡 Solution Overview
- `Multiply2Matrices()` multiplies each corresponding pair of cells: `Result[i][j] = M1[i][j] * M2[i][j]`

## 🔑 Key Concepts
- Element-wise operations between two matrices
- `printf` formatted output

## 📥 Sample Output
```
Matrix 1:      Matrix 2:      Results:
2  5  7        3  1  4        6   5  28
...
```
