# Problem 17: Number Exists In Matrix

## 📋 Description
Checks whether a user-entered number exists anywhere in a 3x3 matrix.

## 💡 Solution Overview
- `IsNumberInMatrix()` returns `true` as soon as a match is found (early exit)

## 🔑 Key Concepts
- Early return for efficiency
- Linear search

## 📥 Sample Output
```
Please Enter the Number to look for in Matrix ? 20
Yes it is there.
```
