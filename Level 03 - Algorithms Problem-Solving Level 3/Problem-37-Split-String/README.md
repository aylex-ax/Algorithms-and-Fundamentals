# Problem 37: Split String

## 📋 Description
Splits a string into individual words and stores them in a vector, then prints the tokens.

## 💡 Solution Overview
- `SplitString()` generalizes the find/substr/erase pattern by pushing each word into a `vector<string>` instead of printing/counting

## 🔑 Key Concepts
- `vector<string>`
- Building a reusable tokenizer function

## 📥 Sample Output
```
Tokens = 3
Ahmed
Yaser
Metwalli
```
