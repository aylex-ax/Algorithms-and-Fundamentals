# Problem 10: Sum of Matrix

## 📋 Description
Fills a 3x3 matrix with random numbers and calculates the total sum of all its elements.

## 💡 Solution Overview
- `SumOfMatrix()` iterates over every cell and accumulates the total

## 🔑 Key Concepts
- Nested loop accumulation
- Note: the inner loop bound uses `Rows` instead of `Columns` — works here only because the matrix is square (3x3)

## 📥 Sample Output
```
Sum of Matrix1 is : 315
```
