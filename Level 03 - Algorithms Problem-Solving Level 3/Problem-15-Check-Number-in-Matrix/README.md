# Problem 15: Check Number in Matrix

## 📋 Description
Counts how many times a user-entered number appears inside a 3x3 matrix.

## 💡 Solution Overview
- `CountNumberInMatrix()` loops through every cell and increments a counter on a match

## 🔑 Key Concepts
- User input (`cin`)
- Linear search over a 2D array

## 📥 Sample Output
```
Enter the Number to Count in Matrix ? 2
Number 2 Count in Matrix is 3
```
