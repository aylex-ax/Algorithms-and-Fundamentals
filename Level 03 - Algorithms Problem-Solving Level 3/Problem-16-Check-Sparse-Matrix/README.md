# Problem 16: Check Sparse Matrix

## 📋 Description
Checks whether a given 3x3 matrix is "sparse" — meaning at least half its elements are zero.

## 💡 Solution Overview
- `IsSparceMatrix()` reuses `CountNumberInMatrix()` to count zeros and compares against half the matrix size

## 🔑 Key Concepts
- Function reuse
- Threshold-based boolean checks

## 📥 Sample Output
```
Yes: It is Sparce
```
