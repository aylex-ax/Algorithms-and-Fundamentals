# Problem 33: Count Vowel

## 📋 Description
Reads a string and counts how many vowels it contains.

## 💡 Solution Overview
- `CountVowels()` loops through the string, using `IsVowel()` (same logic as Problem 32) to test each character

## 🔑 Key Concepts
- Reusing a single-character check across a full string
- Counting pattern

## 📥 Sample Output
```
Number of Vowels is : 4
```
