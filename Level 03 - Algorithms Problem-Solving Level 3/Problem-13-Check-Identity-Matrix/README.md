# Problem 13: Check Identity Matrix

## 📋 Description
Checks whether a given 3x3 matrix is an identity matrix (1s on the diagonal, 0s elsewhere).

## 💡 Solution Overview
- `IsIdentityMatrix()` checks `i == j` cells equal 1 and all other cells equal 0

## 🔑 Key Concepts
- Diagonal detection (`i == j`)
- Boolean-returning validation functions

## 📥 Sample Output
```
Yes: Matrix is Identity
```
