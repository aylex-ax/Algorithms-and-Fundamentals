# Problem 1: 3x3 Random Matrix

## 📋 Description
A C++ program that fills a 3x3 matrix with random numbers between 1 and 100 and prints it.

## 💡 Solution Overview
- `RandomNumber()` generates a random number in a given range
- `FillMatrixWithRandomNumber()` fills the matrix using nested loops
- `PrintMatrix()` displays the matrix in a formatted grid

## 🔑 Key Concepts
- 2D arrays
- Nested loops
- `rand()` / `srand()`
- `setw()` for formatted output

## 📥 Sample Output
```
The Following is a 3x3 Random Matrix :
 45  12  78
 33  90   5
 61  24  17
```
