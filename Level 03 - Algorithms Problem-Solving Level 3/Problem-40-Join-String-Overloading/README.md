# Problem 40: Join String (Overloading)

## 📋 Description
Extends Problem 39 by adding a second `JoinString()` overload that works on a plain C-style string array instead of a vector.

## 💡 Solution Overview
- One `JoinString()` overload takes a `vector<string>&`
- A second `JoinString()` overload takes a `string[]` and a length

## 🔑 Key Concepts
- Function overloading
- Same function name, different parameter types/signatures

## 📥 Sample Output
```
Vector after Join :
Ahmed Yaser Metwalli Mohamed Mohamed Daoud

Array after Join :
Mercedes-Benz-W140-1992
```
