# Problem 4: Sum Each Col in Matrix

## 📋 Description
Fills a 3x3 matrix with random numbers, then calculates and prints the sum of each column.

## 💡 Solution Overview
- `ColSum()` sums the elements of a single given column
- `PrintEachColSum()` loops over all columns and prints each column's sum

## 🔑 Key Concepts
- 2D arrays
- Column-wise iteration (fixed column, varying row)

## 📥 Sample Output
```
Col 1 Sum = 95
Col 2 Sum = 130
Col 3 Sum = 85
```
