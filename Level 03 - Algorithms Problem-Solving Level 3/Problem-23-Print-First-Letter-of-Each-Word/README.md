# Problem 23: Print First Letter of Each Word

## 📋 Description
Reads a string and prints only the first letter of every word in it.

## 💡 Solution Overview
- `PrintFirstLetters()` tracks a boolean flag that becomes true right after a space, marking the next character as a word's start

## 🔑 Key Concepts
- Single-pass string scanning
- State flag to detect word boundaries

## 📥 Sample Output
```
Please enter yor string ? Ahmed Yaser Metwalli
First Letters of this string :
A   Y   M
```
