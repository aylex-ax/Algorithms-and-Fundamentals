# Problem 2: Sum Each Row in Matrix

## 📋 Description
Fills a 3x3 matrix with random numbers, then calculates and prints the sum of each row.

## 💡 Solution Overview
- `RowSum()` sums the elements of a single given row
- `PrintEachRowSum()` loops over all rows and prints each row's sum

## 🔑 Key Concepts
- 2D arrays
- Passing arrays to functions
- Nested loops

## 📥 Sample Output
```
Row 1 Sum = 120
Row 2 Sum = 88
Row 3 Sum = 102
```
