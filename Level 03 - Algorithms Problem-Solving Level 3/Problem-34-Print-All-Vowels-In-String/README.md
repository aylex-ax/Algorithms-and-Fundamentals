# Problem 34: Print All Vowels In String

## 📋 Description
Reads a string and prints every vowel found in it.

## 💡 Solution Overview
- `PrintVowels()` loops through the string and prints each character that passes `IsVowel()`

## 🔑 Key Concepts
- Filtering characters during iteration

## 📥 Sample Output
```
Vowels in string are : a    e    a
```
