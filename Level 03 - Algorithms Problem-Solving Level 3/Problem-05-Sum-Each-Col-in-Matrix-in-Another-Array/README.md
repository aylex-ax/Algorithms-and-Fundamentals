# Problem 5: Sum Each Col in Matrix in Another Array

## 📋 Description
Same as Problem 4, but the column sums are stored in a separate 1D array instead of being printed directly.

## 💡 Solution Overview
- `SumMatrixColsInArray()` fills a 1D array with each column's sum
- `PrintColsSumArray()` prints the resulting array

## 🔑 Key Concepts
- 2D to 1D array mapping
- Function composition (reusing `ColSum`)

## 📥 Sample Output
```
Col 1 Sum = 95
Col 2 Sum = 130
Col 3 Sum = 85
```
