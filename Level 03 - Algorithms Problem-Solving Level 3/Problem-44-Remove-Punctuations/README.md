# Problem 44: Remove Punctuations

## 📋 Description
Removes all punctuation characters from a string.

## 💡 Solution Overview
- `RemoveAllPuncuationsFromString()` builds a new string, copying only characters that are not punctuation according to `ispunct()`

## 🔑 Key Concepts
- `ispunct()`
- Building a filtered copy instead of modifying in place

## 📥 Sample Output
```
Puncuations Removed :
My Name is Ahmed Yaser I am 19 years old I am learning C
```
