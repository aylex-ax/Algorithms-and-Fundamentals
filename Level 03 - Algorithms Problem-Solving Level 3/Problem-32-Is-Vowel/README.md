# Problem 32: Is Vowel

## 📋 Description
Reads a character and checks whether it is a vowel (a, e, i, o, u), regardless of case.

## 💡 Solution Overview
- `IsVowelLetter()` lowercases the input first, then compares it against the 5 vowels

## 🔑 Key Concepts
- Normalizing input before comparison
- Boolean OR chains

## 📥 Sample Output
```
Please enter a Character ? E
Yes Letter 'E' is Vowel.
```
