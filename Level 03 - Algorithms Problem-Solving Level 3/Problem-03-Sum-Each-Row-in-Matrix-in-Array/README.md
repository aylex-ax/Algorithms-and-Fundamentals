# Problem 3: Sum Each Row in Matrix in Array

## 📋 Description
Same as Problem 2, but the row sums are stored in a separate 1D array instead of being printed directly.

## 💡 Solution Overview
- `SumMatrixRowsInArray()` fills a 1D array with each row's sum
- `PrintRowsSumArray()` prints the resulting array

## 🔑 Key Concepts
- 2D to 1D array mapping
- Function composition (reusing `RowSum`)

## 📥 Sample Output
```
Row 1 Sum = 120
Row 2 Sum = 88
Row 3 Sum = 102
```
