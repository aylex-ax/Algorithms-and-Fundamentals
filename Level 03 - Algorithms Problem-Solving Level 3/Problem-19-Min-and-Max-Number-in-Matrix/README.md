# Problem 19: Min and Max Number in Matrix

## 📋 Description
Finds and prints the smallest and largest numbers in a 3x3 matrix.

## 💡 Solution Overview
- `MinNumberInMatrix()` / `MaxNumberInMatrix()` both start from `Matrix[0][0]` and update while scanning all cells

## 🔑 Key Concepts
- Min/max tracking pattern
- Initializing the accumulator with the first element

## 📥 Sample Output
```
Minimum Numbers is : 3
Maximum Numbers is : 77
```
