# Problem 30: Count Letters

## 📋 Description
Reads a string and a character, then counts how many times that exact character appears in the string.

## 💡 Solution Overview
- `CountOneLetter()` loops through the string comparing each character to the target

## 🔑 Key Concepts
- Character comparison
- Simple linear counting

## 📥 Sample Output
```
Please enter yor string ? Ahmed Yaser
Please enter a Character ? a
Letter 'a' Count = 1
```
