# Problem 9: Print Middle Row and Col of Matrix

## 📋 Description
Fills a 3x3 matrix with random numbers, then prints the middle row and the middle column.

## 💡 Solution Overview
- `PrintMiddleRowOfMatrix()` prints row at index `Rows/2`
- `PrintMiddleColOfMatrix()` prints column at index `Columns/2`

## 🔑 Key Concepts
- Integer division to find the middle index
- Fixed-row / fixed-column iteration

## 📥 Sample Output
```
Middle Row of Matrix1 is :
22 20 06

Middle Col of Matrix1 is :
05 20 03
```
