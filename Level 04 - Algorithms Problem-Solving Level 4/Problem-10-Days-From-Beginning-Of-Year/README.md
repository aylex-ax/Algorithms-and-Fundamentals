# Problem 10: Days from the Beginning of Year

## 📋 Description
Given a full date, calculates how many days have passed since the start of that year (i.e. the date's day-of-year order).

## 💡 Solution Overview
- `NumberOfDaysFromTheBeginingOfTheYear()` sums the days of every month before the given month, then adds the day itself

## 🔑 Key Concepts
- Accumulating totals across a loop of variable length
- Building block for later problems (day-order ↔ date conversion)

## 📥 Sample Output
```
Number of Days from the begining of the Year is 201
```
