# Problem 14: Date1 Equals to Date2

## 📋 Description
Compares two dates and reports whether they represent the exact same day.

## 💡 Solution Overview
- `IsDate1EqualToDate2()` checks year, month, and day all match, using nested ternary expressions

## 🔑 Key Concepts
- Struct field-by-field equality checking
- Same nested-ternary style as Problem 13

## 📥 Sample Output
```
Yes, Date1 is Equal to Date2.
```
