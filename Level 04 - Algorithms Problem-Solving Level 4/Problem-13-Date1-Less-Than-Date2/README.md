# Problem 13: Date1 Less Than Date2

## 📋 Description
Compares two dates and reports whether the first is chronologically before the second.

## 💡 Solution Overview
- `IsDate1BeforeThanDate2()` compares year first, then month, then day, using nested ternary operators

## 🔑 Key Concepts
- Multi-field comparison (year → month → day priority)
- Nested ternary expressions as a compact alternative to if/else chains

## 📥 Sample Output
```
Yes, Date1 is Less than Date2.
```
